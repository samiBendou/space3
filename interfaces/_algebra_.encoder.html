Interface Encoder

Introduction

Encodes values in different native Javascript types.

Example

u.string(); // "(ux, uy, uz)"
m.array(); // [mxx, myx, mzx, mxy, ... ]

Float64Array

Encoder class inherits from Float64Array in order to provide double precision computation, an array access u[k]. and native C/C++ array compatibility.
